#  **Code Explanation (For Absolute Beginners)**

Here is the entire `index.js` file with simple comments:

```javascript
const express = require('express'); // Import the express framework
const app = express();              // Create an express application

// Create a GET API endpoint at the homepage '/'
app.get('/', (req, res) => {
  res.json({ message: 'Hello World from Express!' });
});

// Choose a port for your server
const PORT = 3000;

// Start the server and listen for requests
app.listen(PORT, () => {
  console.log(`Server running on http://localhost:${PORT}`);
});
```

This is all you need to build your first backend!

#  **Common Issues & Fixes (Troubleshooting)**

Here are the most common errors beginners face:

---

###  1. `Error: Cannot find module 'express'`

**Fix:**

```bash
npm install express
```

---

###  2. Port already in use

**Fix:**

Edit `PORT` or run:

```bash
npx kill-port 3000
```

---

###  3. Server doesn‚Äôt restart automatically

**Fix: Install nodemon**

```bash
npm install -D nodemon
```

Then run:

```bash
npx nodemon index.js
```

---

###  4. Browser shows blank page

**Fix:** Visit the correct URL:

üëâ [http://localhost:3000](http://localhost:3000)
